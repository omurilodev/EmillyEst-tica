Como obter a URL do Apps Script:

  1. Na planilha → Extensions → Apps Script
  2. Cole o código do Code.gs e salve
  3. Execute setupSheet() uma vez (selecione no dropdown)
  4. Deploy → New deployment → Web App
  5. Execute as: Me | Access: Anyone
  6. Copie a URL gerada (termina com /exec)
A URL deve terminar com /exec
